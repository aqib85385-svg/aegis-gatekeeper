# Developer Manual

## Directory Organization
*   `src/`: Client view layouts, custom components, client services.
*   `src/workers/`: Multi-threaded Canvas compression routines.
*   `api/`: Backend serverless proxy routers and helper utilities.
*   `tests/`: Unit, integration, component, and E2E test suites.

## Scripts
*   `npm run dev`: Start local client server.
*   `npm run build`: Compile static production bundles.
*   `npm run lint`: Run oxlint static code checks.
*   `npm run test`: Execute Vitest test runner.
