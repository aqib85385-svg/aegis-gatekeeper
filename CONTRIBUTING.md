# Contributing Guidelines

Welcome to the Aegis GateKeeper project! 

## Workspace Setup

1. Fork the repository and check out your feature branch.
2. Install local node dependencies:
   ```bash
   npm install
   ```
3. Run the development server locally:
   ```bash
   npm run dev
   ```

## Verification Expectations

Before submitting any Pull Request, ensure that:
*   Linting checks pass: `npm run lint`
*   Automated tests run clean: `npm run test`
*   Production build completes without errors: `npm run build`
